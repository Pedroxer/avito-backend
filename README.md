# Тестовое задание на backend в Avito

В проекте используются
- gin-framework
- migrate (для применения миграций)
- sqlc (для генерации запросов к бд)

## Чтобы запустить
Загрузить с github:
``` git clone git@github.com:Pedroxer/avito-backend.git ```

### Запуск:
Выполнить в директории
```
docker compose up
 ```
